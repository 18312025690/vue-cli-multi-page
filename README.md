# vue-cli多页面

> A Vue.js project，[demo地址][http://v.lanchenglv.com/demo/vue-cli-multi-page/module/login.html]



## Build Setup

``` bash
# 安装
npm install

# serve with hot reload at http://localhost:8083/module/login.html
npm run dev

# build for production with minification
npm run build


```
